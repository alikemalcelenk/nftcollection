# nftcollection

Example Project that is written in JavaScript using Vue, Vuex.

## Installation and Setup Instructions

#### Installation:

```
yarn install
```

#### To start server:

```
yarn serve
```

#### To visit app:

`localhost:8080`

## Tech Stack

- vue
- vuex
- eslint
- prettier
